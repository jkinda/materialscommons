Overview

Materials Commmons allows researchers to deposit, search and access data. The data can be marked as public or private, you can control access by group, and you can attribute the data as needed. The site provides a simple set of tools for uploading, searching and exploring your data. Because data sets may be large in size the site also provides tools for uploading and downloading data sets offline. This allows you to start a large upload or download and then forget about it. When the data has been successfully uploaded it will appear in your list of data.

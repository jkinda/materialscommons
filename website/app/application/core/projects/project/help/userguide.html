  • Glossary

    Glossary

    Dashboard:
    is the area where you can see all panels for a project.
    Draft:
    can be defined as an incomplete provenance step. When you save your provenance as draft you can go back and modify at anytime.
    Last login:
    will provide your last login date onto Materials Commons.
    Project:
    is defined as a workspace where you can store all the information related to a project or perform some actions against them. Every project will have its own defined workspace.
    Provenance:
    is defined as the history of data
    Process:
    is one of the key step in provenance creation. Process is defined as an experiment / computation / data analysis with a set of inputs and outputs.
    Panel:
    is a container to list various items (such as reviews/samples/notes/processes/files) inside a project.
    Review:
    is defined as an assessment or examination of something with the possibility or intention of instituting change if necessary.
    Sample:
    is defined as a material on which user can perform some experiment. During provenance creation sample will be an input into the process.
    Template:
    is a placeholder for detailed steps inside a process(experiment/computation/data analysis). There are set of pre-defined templates on Materials Commons.
    Sideboard:
    is an area where you can add/drag various items to compare side-by-side.

    Navigation:
    Draft

    MaterialsCommons navigation bar helps user to switch between different projects and navigate between global pages.

    Settings:

    Apart from navigation user can also update their username, password, customize templates( for provenance) and reset apikey.

    Help

    Help page or Userguide is a manual which describes about what and how to use MaterialsCommons. For the ease of use, userguide is designed such that it can be opened on the side, while the user can still work on their respective panels.

  • Home Page

    Home Page

    Homepage contains panels for Reviews, Samples, Files, Notes and Provenance. Content inside panels changes with respect to the project. In this section we will describe about commonly used icons and their function.

    Draft
  • Reviews

    Reviews

    Panel will list all the reviews created inside a project. List may contain reviews which you have requested to others or vice-versa. When you click on the title, review toggles and details will be opened. You can comment on the review by clicking on the comment button. If you close the review it will be removed from the list (archived).

    Draft
    How to create a review

    When you click create button it will open up a form to the right where you can fill in the title, pick a user who has to review and add your comment.

  • Samples

    Samples

    Panel will list all the samples inside a project. When you click on the sample name you can see more information about the sample.

    Draft
  • Processes

    Provenance

    Provenance panel will list all the processes created for a project.

    Draft
    How to create a provenance:

    Step1 : Choose Template

    In order to create provenance first you have to choose a template. Based on the template you choose, provenance steps may vary. For Example: You wanted to collect data through Atom Probe Experiment. So now you have to choose APT Data Collection Template from the drop-down list

  • Files

    Files

    Draft

    Files panel will show you all your uploaded files for a project. It will preserve the directory layout. You can view the file by clicking on the name of the file.

    How to create a sub folder

    Project name will be your root folder. Any folder's you wanted to create will be under this root folder. Click on the plus icon next to the folder name and then fill out the details and click add.

    How to upload a file

    If you want to upload a file into a directory then click on the upload icon next to the folder into which you wanted to upload the file. Then a pop up window will appear through which you can browse and select a file. After you select it will appear under "Files to Upload List" as shown in the image below.

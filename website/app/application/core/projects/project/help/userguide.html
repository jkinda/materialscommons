  • Nav bar

    Navigation Bar

    Navigation:
    Draft

    MaterialsCommons navigation bar helps user to navigate between global pages.

    Settings:

    Apart from navigation user can also update their username, password, customize templates( for provenance) and reset apikey.

    Global Reviews
    Draft

    Global reviews section will list all the active reviews from all the projects in your list. Global reviews are categorized into 2 types. 1) Requested 2) To Conduct.
    Requested reviews are the reviews you have created and assigned to other users. Whereas To Conduct reviews are the reviews which are assigned to you by other users. Detailed information about the reviews are discussed in reviews section. When you click on a particular review it will take you to the respective project under which the review is created.

    Help

    Help page or Userguide is a manual which describes about what and how to use MaterialsCommons. For the ease of use, userguide is designed such that it can be opened on the side, while the user can still work on their respective panels.

  • Project Bar

    Project Bar

    Switch Projects

    Projects bar indicates total number of projects user has created/shared. Dropdown menu will give you the list of project names. When you click on project name it will take you to the respective Homepage.

    Draft
    Create New Project

    To Create a new project click on the drop-down menu and select button called "New Project" and enter the new project name.

    Bottom of the project taskbar has icons for 1) Project Setting 2) Calendar 3) Panels

    Project Settings

    Project settings help project owner to give permission/access to view their project. Owner can add/remove the users depending on with whom he/she wanted to share their project. Currently When a project is shared everything inside the project is shared.

    Calendar
    Draft

    Calendar helps user to keep track of the items created and how many of them were created. You can click on a particular day to filter out all the items created/updated on that day.

    Panels

    Panels refer to sideboard, Reviews, Files, Samples, Notes, Processes. More information about these panels are discussed under respective sections of userguide.

  • Home Page

    Home Page

    Homepage contains panels for Reviews, Samples, Files, Notes and Provenance. Content inside panels changes with respect to the project. In this section we will describe about commonly used icons and their function.

    1. Add to Sideboard

    This icon helps to compare various items. When you click item will be added to the board. After you add all the items split the sideboard such that you can compare items side-by-side.

    Draft
    2. Search

    Search will filter out all the items matching the entered search name.

  • Reviews

    Reviews

    Reviews panel will list all the reviews belonging to a project. List may contain reviews which you have requested to others or vice-versa. User will identify the review by title.

    Draft

    When you click on the title, review toggles and details will be opened. You can comment on the review by clicking on the comment button. If you close the review it will be removed from the list.

  • Samples

    Samples

    Samples panel will list all the samples belonging to a project.

    Draft

    When you click on the sample name you can see more information about the sample.

  • Notes

    Notes

    Notes panel will list all the notes belonging to a project. User can create a notes for project/sample/file. Note creation is discussed more in detail under section How to create Note

    Draft

    When you click on the note title you can view detailed notes.

  • Provenance

    Provenance

    Provenance panel will list all the processes created for a project.

    Draft

    When you click on the process name, process details will be opened. Graph will briefly describe about what are the inputs to the process and outputs from the process.

  • File layout

    Files

    Draft

    Files panel will show you all your uploaded files for a project. It will preserve the directory layout. You can view the file by clicking on the name of the file.

    Global Reviews
    Draft

    Global reviews section will list all the active reviews from all the projects in your list. Global reviews are categorized into 2 types. 1) Requested 2) To Conduct.
    Requested reviews are the reviews you have created and assigned to other users. Whereas To Conduct reviews are the reviews which are assigned to you by other users. Detailed information about the reviews are discussed in reviews section. When you click on a particular review it will take you to the respective project under which the review is created.

    Help

    Help page or Userguide is a manual which describes about what and how to use MaterialsCommons. For the ease of use, userguide is designed such that it can be opened on the side, while the user can still work on their respective panels.

  • Reviews

    Reviews

    Panel will list all the reviews created inside a project. List may contain reviews which you have requested to others or vice-versa. When you click on the title, review toggles and details will be opened. You can comment on the review by clicking on the comment button. If you close the review it will be removed from the list (archived).

    Draft
    How to create a review

    When you click Plus icon it will open up a form where you can fill in the title, pick a user who has to review and add your comment.

    Attach samples/files/notes to review

    There is an option to attach samples/files/notes along with your comment. In order to attach click on the + sign of the respective item which will trigger and open up an option to add those items to your review. Example: If you want to attach a note, click on the + sign next to notes and then you will see options in the notes panel as shown in picture below. When you hit create it will create a review for a project and will be notified to the assignee under global-reviews( which is in navbar).

  • Samples

    Samples

    Panel will list all the samples inside a project. When you click on the sample name you can see more information about the sample.

    Draft
    How to create a sample

    When you click Plus icon it will open up a form where you can fill in the name, alloy supplier and sample type. Icons annotated as 1, 2 and 3 (Image) are explained below:

    1. Composition

    When you click composition button, it will open up dropdown list of elements using which you can add composition to the sample

    2. Share across projects

    sample can be shared across various projects. So this icon helps you to associate sample across different projects.

    3. Sample note

    This icon helps you to add a notes to the sample which will appear under "Notes Panel"

    After you enter all the information then hit create which will add your new sample to the list.

  • Notes

    Notes

    List:

    Panel will list all the notes created inside a project. A note can be created for a project/sample/file. When you click on the note title you can view detailed message.

    Draft
    How to create a note

    When you click Plus icon it will open up a form where you can fill in the title and detailed message in the note area. You can attach images/links and format your text. When you hit create it will create a note for a project and it will then appear on your list.

    Note creation for sample and file will be discussed under respective sections.

  • Processes

    Provenance

    Provenance panel will list all the processes created for a project.

    Draft

    When you click on the process name, process details will be opened. Graph will briefly describe about what are the inputs to the process and outputs from the process.

    How to create a provenance:

    Based on the template you choose, provenance steps may vary. In general we can classify the steps as:

    1. Process

    Process (experiment/computation): Choose a template from the dropdown list and hit start. Now enter the name of the process and other details and move on to the next step which is inputs

    2. Inputs 3. Outputs
  • Files

    Files

    Draft

    Files panel will show you all your uploaded files for a project. It will preserve the directory layout. You can view the file by clicking on the name of the file.

    How to create a sub folder

    Project name will be your root folder. Any folder's you wanted to create will be under this root folder. Click on the plus icon next to the folder name and then fill out the details and click add.

    How to upload a file

    If you want to upload a file into a directory then click on the upload icon next to the folder into which you wanted to upload the file. Then a pop up window will appear through which you can browse and select a file. After you have selected it will appear under "Files to Upload List which is above the project tree.

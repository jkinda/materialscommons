  • Glossary

    Glossary

    Datafile:
    Datafiles are the raw files containing information from experiments or computations.
    Project:
    is defined as a workspace where you can store all the information related to a project or perform some actions against them. Every project will have its own defined workspace.
    Provenance:
    is defined as the history of data.The provenance information that describes how datafiles and samples are created, transformed, analyzed or measured is stored in a Process
    Process:
    is one of the key step in provenance creation. Process is defined as an experiment / computation / data analysis with a set of inputs and outputs.
    Panel:
    is a container to list various items (such as reviews/samples/notes/processes/files) inside a project.
    Review:
    is defined as an assessment or examination of something with the possibility or intention of instituting change if necessary.
    Sample:
    Samples are digital objects representing experimental or virtual computational materials samples. Key attributes are stored in Samples so that they can be easily used for analysis and modeling
    Template:
    is a placeholder for detailed steps inside a process(experiment/computation/data analysis). There are set of pre-defined templates on Materials Commons.

  • Home Page

    Home Page

    Homepage contains panels for Reviews, Samples, Files, Notes, Provenance and Calendar will keep track of all your events. Content inside panels changes with respect to the project.

    Draft
  • Reviews

    Reviews

    Panel will list all the reviews created inside a project. List is categorized into 4 types:
    MY REVIEWS: are the reviews you have requested others
    DUE: are the reviews requested to you by other researcher
    ALL: MY REVIEWS + DUE
    ARCHIVED: are the closed reviews

    Draft
    How to create a review

    When you click create button it will open up a form to the right where you can fill in the title, pick a user/users who has to review and add in your comment.

  • Samples

    Samples

    Panel will list all the samples inside a project. When you click on the sample name you can see more information about the sample.

  • Files

    Files

    Draft

    Files panel will show you all your uploaded files for a project. It will preserve the directory layout. You can view the file by clicking on the name of the file.

    How to create a sub folder

    Project name will be your root folder. Any folder's you wanted to create will be under this root folder. Click on the CREATE FOLDER link, enter name of the folder and click add.

    How to upload a file

    If you want to upload a file into a directory then click on the upload icon Then a pop up window will appear through which you can browse and select a file. After you select it will appear under "Files to Upload List" as shown in the image below.

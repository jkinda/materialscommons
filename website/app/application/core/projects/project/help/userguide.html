Userguide
  • Nav bar

    Navigation Bar

    Navigation:

    MaterialsCommons navigation bar helps user to navigate between global pages.

    Settings:

    Apart from navigation user can also update Username, password, customize templates and reset apikey.

    Global Reviews:

    Global reviews section will show you all the reviews irrespective of what project you are in. Global reviews are categorized into 2 types. 1) Requested 2) To Conduct. Requested reviews are the reviews you have created and assigned to other users. Whereas To Conduct reviews are the reviews which are assigned to you by other users. Detailed information about the reviews are discussed in reviews section.

  • Project Bar

    Project Bar

    Switch Projects

    Projects task bar indicates total number of projects user is associated with. Also user can switch between projects.

    Create New Project

    To Create a new project click on the drop-down menu and select button called "New Project" and enter the new project name.

    Bottom of the project taskbar has icons for 1) Project Setting 2) Calendar 3) Panels

    Project Settings

    Project settings help project owner to give permission/access to view their project. Owner can add/remove the users depending on with whom he/she wanted to share their project. Currently When a project is shared everything inside the project is shared.

    Calendar

    Calendar helps user to keep track of the items created and how many of them were created. You can click on a particular day to filter out all the items created/updated on that day.

    Panels

    Panels refer to sideboard, Reviews, Files, Samples, Notes, Processes. More information about these panels are discussed under respective sections of userguide.

  • Reviews

    Reviews

    Reviews panel will list all the reviews belonging to a project. List may contain reviews which you have requested to others or vice-versa. User will identify the review by title.

    When you click on the title, review toggles and details will be opened. You can comment on the review by clicking on the comment button. If you close the review it will be removed from the list.

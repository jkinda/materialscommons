Create Experiment From Spreadsheet

Spreadsheet layout

Spreadsheets can be ingested and used to create a workflow and load samples and measurements. The first column must be the name of the sample. The second column is optional and contains the name of the worksheet that is the process that this sample came from. The remaining rows are process and sample attributes. Sample attributes are designated with a "S:" or "s:", process attributes are designated with a "P:" or "p:". If you don't designate a column then it is assumed to be a sample attribute.

Units are specified in the header at the end of the header by enclosing them in paranthesis. For example "s:grain size(mm)" would designate a sample attribute named "grain size" with units in "mm".

If you have a parent column then each of your sample names must be unique in a worksheet. If you do not have a parent column then this restriction doesn't apply.

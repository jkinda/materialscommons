ETL - Create Experiment using direct upload

Create an experiment, using Globus to upload an Excel spreadsheet and its data files. If you do not wish to use Globus at this time, you can switch to an alternate dialog.

To use this form, you will first need to create a shared Globus endpoint and give read permission to the Materials Client broker: MaterialsCommonsTest-ConfClient.


To create an experiment without uploading files, fill out the form and drag/drop (or open) the Excel spreadsheet, below.

To create an experiment with the data files uploaded, using Globus, switch to the alternate dialog.
